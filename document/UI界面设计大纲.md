## 整体设计理念

### 核心设计原则

**1. 专注与效率优先**
- 学习工具应该帮助用户专注,而不是分散注意力
- 减少不必要的动效和装饰,快速进入学习状态
- 核心操作不超过3步

**2. 清晰的信息层级**
- 重要信息优先:今日待办 > 学习进度 > 历史数据
- 使用视觉权重区分主次信息
- 避免信息过载,适当留白

**3. 即时反馈与激励**
- 每个学习行为都有明确的反馈
- 进度可视化,让成就感看得见
- 温和的激励而非过度游戏化

**4. 符合学习场景**
- 单手操作友好(通勤时使用)
- 夜间模式支持(晚上学习)
- 快速切换任务(碎片化学习)

## 视觉风格定义

### 色彩系统

**主色调 - 沉稳蓝**
```
主色: #4A90E2 (传递专业、理性、信任感)
辅助色: #5FB3F6 (浅蓝,用于次要按钮)
深色: #2C5F8D (用于重要文字)
```

**功能色**
```
成功/完成: #52C41A (绿色)
警告/待办: #FAAD14 (橙色)
错误/遗忘: #F5222D (红色)
AI助手: #722ED1 (紫色,体现智能感)
```

**中性色**
```
背景: #F7F9FC (浅灰蓝,护眼)
卡片: #FFFFFF
文字主要: #1A1A1A
文字次要: #8C8C8C
分割线: #E8E8E8
```

### 字体系统

```
标题字体: PingFang SC / San Francisco (系统字体)
正文字体: 同上

字号规范:
- 大标题: 24px (页面主标题)
- 中标题: 18px (卡片标题)
- 正文: 16px (主要内容)
- 辅助文字: 14px (时间、标签)
- 小字: 12px (提示信息)
```

### 圆角与间距

```
圆角:
- 卡片: 12px
- 按钮: 8px
- 小组件: 6px

间距:
- 页面边距: 16px
- 卡片间距: 12px
- 组件内边距: 12-16px
- 小元素间距: 8px
```

## 关键交互流程设计

### 流程1: 新用户引导

```
欢迎页 → 选择学习目标 → 评估当前水平 → 设置学习时间 → AI生成计划 → 开始学习
```

**引导设计**:
- 3-4步完成,不超过1分钟
- 每步都有明确的进度指示
- 可跳过,但建议完成
- 使用插画增强亲和力

### 流程2: 每日学习循环

```
打开App → 查看今日任务 → 选择任务 → 完成学习 → 获得反馈 → 继续/结束
```

**设计要点**:
- 首页直接展示"立即复习"按钮
- 学习过程中减少干扰
- 完成后即时反馈和激励
- 支持快速切换任务

### 流程3: AI答疑交互

```
遇到问题 → 打开AI助手 → 输入问题 → AI回答 → 追问/生成卡片 → 结束对话
```

**设计要点**:
- 全局可访问(悬浮按钮)
- 支持文字/语音/图片输入
- 回答支持语音朗读
- 可一键生成记忆卡片

## 组件设计系统

### 基础组件

**1. 按钮**
```
主要按钮: 大尺寸, 主色调, 圆角8px
次要按钮: 中尺寸, 描边, 圆角8px
文字按钮: 纯文字, 主色调
危险按钮: 红色, 用于删除等操作
```

**2. 卡片**
```
标准卡片: 白色背景, 圆角12px, 投影subtle
高亮卡片: 主色调背景或渐变, 用于强调
小卡片: 紧凑布局, 用于列表项
```

**3. 进度条/环**
```
线性进度条: 高度4px, 圆角2px, 带动画
环形进度: 用于目标完成度, 中间显示百分比
分段进度: 多阶段目标, 用点连接
```

**4. 状态标签**
```
完成: 绿色背景
进行中: 橙色背景
未开始: 灰色背景
尺寸: 小号, 圆角4px
```

### 特殊组件

**1. 学习卡片 (记忆卡)**
- 类似实体卡片的视觉设计
- 支持翻转动画
- 可自定义背景和字体大小

**2. 日历热力图**
- GitHub风格
- 点击查看当天详情
- 不同深度表示学习强度

**3. 统计图表**
- 使用库: Recharts
- 简洁配色,清晰易读
- 支持点击交互查看详情

## 响应式与适配

### 屏幕适配

**支持尺寸**:
- 小屏 (iPhone SE): 375px
- 标准屏 (iPhone 13): 390px
- 大屏 (iPhone 14 Pro Max): 430px
- 折叠屏 / iPad: 响应式布局

**适配策略**:
- 使用相对单位 (rem/em)
- 关键元素使用百分比宽度
- 大屏增加内容密度,小屏简化展示
- 横屏优化(如复习卡片更大)

### 主题模式

**浅色模式** (默认)
- 清爽明亮,适合白天使用
- 背景: #F7F9FC

**深色模式**
- 护眼舒适,适合晚上使用
- 背景: #1A1A1A
- 主色调调整为更柔和的色调
- 自动跟随系统或手动切换

## 动效设计原则

### 微动效

**页面过渡**: 
- 使用淡入淡出 + 轻微位移
- 时长: 200-300ms
- 缓动函数: ease-out

**按钮反馈**:
- 点击缩放效果 (scale 0.95)
- 时长: 100ms
- 增强操作确定感

**卡片翻转**:
- 3D翻转效果
- 时长: 400ms
- 增加学习趣味性

**进度更新**:
- 平滑的数字增长动画
- 进度条填充动画
- 增强成就感

### 加载状态

**骨架屏**: 
- 用于列表加载
- 保持布局稳定
- 比转圈更友好

**下拉刷新**:
- 自定义刷新动画
- 可使用品牌元素

**加载失败**:
- 清晰的错误提示
- 提供重试按钮
- 避免技术术语
